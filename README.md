## How to Upload to GitHub

1. Go to [github.com](https://github.com) and sign in
2. Click the **+** button → **New repository**
3. Name it `paolo-kiki-jewelry` (or anything you like)
4. Make it **Public**
5. Click **Create repository**
6. Upload `index.html` and `README.md` by dragging them in, or use the **uploading an existing file** link
7. Click **Commit changes**

## How to Enable GitHub Pages (free hosting)

1. In your repository, go to **Settings**
2. Scroll to **Pages** in the left sidebar
3. Under **Source**, select **Deploy from a branch**
4. Choose **main** branch → **/ (root)**
5. Click **Save**
6. Your site will be live at: `https://YOUR-USERNAME.github.io/paolo-kiki-jewelry`
